# SentinelAI: AI Investigation Dashboard

## Overview

The `ai-investigation` module serves as the primary frontend interface for the SentinelAI AI Copilot. It enables Security Operations Center (SOC) analysts to interact naturally with the AI Decision Intelligence Engine to understand complex attack paths.

## Responsibilities

* **Prompt Interface:** Provides the conversational UI for analysts to query specific machine identities and risk findings.
* **Error Handling:** Gracefully handles and displays structured enterprise error codes (e.g., `AI_AUTH_FAILED`, `AI_RATE_LIMITED`) returned by the backend.
* **Context Display:** Renders the parsed, hallucination-free JSON executive summaries and attack path analyses generated by Gemini 3.5 Flash.

## Architecture

This module heavily relies on React Server Components (where applicable) and client-side state management for the chat interface. It acts as the visual consumer of the `/api/v1/ai` backend routes, translating the AI's deterministic findings into a human-readable threat narrative.
