# SentinelAI: Analytics & Telemetry

## Overview

The `analytics` module offers a macro-level view of system performance and ingestion metrics, separate from the security-focused heuristics.

## Responsibilities

* **Ingestion Metrics:** Visualizes the volume of logs processed by the backend over time.
* **Cost Observability:** (Roadmap) Provides insights into the LLM token usage (Gemini 3.5 Flash) and associated operational costs for the AI Decision Intelligence Engine.
