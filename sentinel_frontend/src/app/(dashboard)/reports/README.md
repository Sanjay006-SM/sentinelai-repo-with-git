# SentinelAI: Reports Module

## Overview

The `reports` module provides the functionality to generate, manage, and download compliance and executive security reports.

## Responsibilities

* **PDF Generation Delivery:** Integrates with the backend's AI summarization pipelines (e.g., `generate_executive_report_summary`) to deliver professional, exportable `executive_report.pdf` documents.
* **Report Scheduling:** (Roadmap) Provides UI for scheduling recurring automated reports for stakeholders.
