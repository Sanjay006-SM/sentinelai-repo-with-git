# SentinelAI: Dashboard Overview

## Overview

The `dashboard` module is the primary landing page for authenticated users, offering a high-level, executive snapshot of the organization's current security posture.

## Responsibilities

* **Telemetry Visualization:** Renders top-level Key Performance Indicators (KPIs), such as the overall Organizational Risk Score, total active threats, and ingestion volume.
* **Recent Activity:** Displays a real-time feed of the most critical recent `RiskFinding` records.
* **Trend Analysis:** Utilizes charting libraries to display historical risk trends over time, providing immediate context on whether the environment's security is improving or degrading.
