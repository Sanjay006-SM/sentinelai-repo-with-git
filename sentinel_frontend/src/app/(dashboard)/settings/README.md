# SentinelAI: Application Settings

## Overview

The `settings` module houses user-specific preferences and general application configuration.

## Responsibilities

* **User Profile:** Manage avatar, name, and personal preferences.
* **Theme Controls:** Integrates with the global design system (e.g., Tailwind dark mode) to toggle visual appearances.
* **Notification Preferences:** Configure alert thresholds for email or webhook integrations based on severity classifications.
