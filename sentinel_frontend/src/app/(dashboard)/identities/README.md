# SentinelAI: Identities Management

## Overview

The `identities` module provides a comprehensive inventory of all discovered machine and human identities (Users, Roles, Service Accounts) synchronized across the hybrid architecture.

## Responsibilities

* **Identity Inventory:** Lists all `MachineIdentity` entities, allowing analysts to search, filter, and sort by calculated risk score.
* **Profile Deep Dive:** Provides a detailed profile view for a single identity, aggregating all associated access logs, heuristic violations, and historical behavior into a single pane of glass.
