# AI Output Verification Layer

## Purpose
Ensures that LLM-generated responses do not contain hallucinations by cross-referencing claims against ground-truth evidence.

## Responsibilities
- Extract factual claims from the AI response.
- Execute secondary LLM calls or deterministic checks to verify claims against the `RiskEvidence`.
- Provide corrected claims or flag unverified statements.

## Inputs
- Draft `AIResponse` generated by the primary analyst node.
- Raw `RiskEvidence` context.

## Outputs
- `VerifiedResponse` schema containing the confidence score and finalized response.

## Dependencies
- Local `schemas.verified_response`

## Future Work
- Build LangGraph nodes that loop the generation process if verification fails severely (self-correction).
